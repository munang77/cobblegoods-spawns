# CobbleGoods 전설 스폰 도감

Cobblemon + Myths and Legends 기준, CobbleGoods 애드온의 **전설·환상 포켓몬 스폰 정보**(바이옴·시간·날씨·확률)를 정리한 페이지입니다.

- 전설 61종 + 환상 20종 (총 81종)
- 2시간 전설 이벤트(파이어/프리져/썬더/뮤/뮤츠) 확률
- 자연 스폰 등급(버킷) 확률 구조
- 이름/분류/조건별 검색·필터

데이터는 모드의 스폰 파일에서 자동 생성됩니다.

> 울트라비스트·패러독스는 현재 스폰 데이터에 없습니다(0종).
